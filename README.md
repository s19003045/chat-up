# Chat-up back-end

Chat-up is chatting application with socket.io

## Getting Started

### Prerequisites

- Node.js: 12.16.1 LTS
  - https://nodejs.org/en/
  - includes npm 6.13.4
- nodemon
  - ```npm install -g nodemon```

### Installing

- git clone project
  - ```git clone https://github.com/s19003045/chat-up.git```
- install packages
  - ```npm install```


- run the project
  - ```npm run dev```
  - you will see "Express app listening on port 3000!" on the terminal
